# Pixel Miku Helper ??

A tiny pixel Hatsune Miku who hangs out on your screen, reacts to keywords, and keeps you company while you browse.

## Features
- Draggable Pixel Miku
- Speech bubble reactions
- Keyword detection (music, anime, homework, etc.)
- Song suggestions
- Right-click chat menu
